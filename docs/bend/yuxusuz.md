# Preface

Rüyaların onsuz(?) da gerçək(?) olmadığını(?) bilirik(?). Amma başlığın iddiası doğruluq statusundan çox ontolojik yoxluqdur.

## Beynimiz bizdən deyil

Nə qədər özümüz bekar olsaqda, beynimiz bekarçılığı və ekzistensial heçliyi sevmir. Data olmadıqda özü üçün yaradır. Bir neçə dəqiqəlik görünənin və eşidənin inkarı[^1] perseptiv illüziya ilə nəticələnir.

[^1]: təzahürsüzlük

## Getməyişat

İnsanlar yatana zaman onlara datanın axını dayanır. Və qeyd etdiyimiz kimi bunun nəticəsində *yuxu* adlandırdıqları şeyləri görürlər.

# Yıxılın yatın

Yatmaq ilə rüya görməyin daxili vəhdətdə olmadığını anladığınız üçün bu gecə(?) rahat yata bilərsiz.
